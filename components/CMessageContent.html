# CMessageContent Message Content

This component represent the hidden row of each of CMessageRow, this component display, args, kwargs and queueName of the asociated Message.

# Props

  props: {
    messageId: String,
    actorName: String,
    colspan: Number
  }

# Data

  data() {
    return {
      args: String,
      kwargs: String
    };
  }
